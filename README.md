# stkATOM WebApp 

### Tools and Technologies

- NextJS
- TypeScript
- ESLint setup with Airbnb config
- Husky git hooks
- Cypress and RTL support for tests

### Requirements

- Node.js 14+ and npm

### Getting started

For the development server:

```shell
npm run dev
```

### Deploy to production

To see the production build locally:

```shell
$ npm run build
$ npm run start
```
